# Ultrasooq Unified 6-Level Category Tree

## Sources & Methodology
- **Alibaba.com**: ~40 top-level B2B/B2C categories (alibaba.com/Products)
- **Amazon.com**: 41 browse nodes, 25,000+ subcategories (browsenodes.com)
- **eBay US**: 34 L1 categories, ~20,000 total (isoldwhat.com, eBay Taxonomy API)
- **Taobao**: 16 main category groups (parcelup.com, aioexpress.com)
- **Oman/ME additions**: frankincense, dates, halwa, oud, dishdasha, kumma, abayas

## Format: L1 > L2 > L3 > L4 > L5 > L6
